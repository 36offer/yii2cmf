## 安装:

进入项目根目录

1. `composer install` 

2. `php yii app`

3. `php yii serve`

4. 收工

    前台访问地址: `http://localhost:8080/`

    后台访问地址: `http://localhost:8080/admin/` 管理员 帐号 `hehe` 密码 `111111`
